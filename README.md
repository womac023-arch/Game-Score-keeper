# Game Score Keeping System

A simple Python program to track scores for multiple players.

*   **Add Players:** Users can add player names to the game.
*   **Track Scores:** The system keeps total scores for each player.
*   **Update Scores:** Scores can be incremented based on game events.
*   **Display Scores:** The current scores for all players can be displayed.

1.  Ensure you have Python installed.
2.  Save the code as a Python file.
3.  Run the script from your terminal.
